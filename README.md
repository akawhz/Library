# 模块目录

当前版本：v1.0\
\
**前后台公有模块：**

1. 用户模块（前后台）
   * 用户登录注册功能
   * 后台用户管理（增删改查）
   * 用户权限管理（角色分配、权限控制）\

   * 权限控制（根据角色权限显示对应菜单）
2. 文章模块（前后台）
   * 文章展示（列表、详情）
   * 文章分类管理（增删改查）
   * 文章标签管理（增删改查）
   * 文章撰写与发布
   * 热门文章展示
   * 按类别获取文章
   * 根据标签检索文章\

3. 友链模块（前后台）
   * 友链展示
   * 友链管理（增删改查）

**前台系统独有模块：**

4\. 评论模块

* 文章评论
* 回复评论
* 发送emoji表情

5. 音乐厅模块
   * 在前台展示推荐单曲，推荐歌单，音乐播放。

**后台系统独有模块：**&#x20;

\
6\. 菜单管理模块

* 子菜单管理（增删改查）

以上是按照模块的涉及范围划分的一个版本设计。在后续的版本迭代中，可以根据需要增加新的模块或对现有模块进行优化和扩展。
